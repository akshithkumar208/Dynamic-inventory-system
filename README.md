# Dynamic Inventory Reallocation System

This project dynamically reallocates inventory between a factory and four retail shops
using demand forecasting and optimization.

## Features
- Big data simulation
- ARIMA demand forecasting
- Stock gap analysis
- Factory → Shop & Shop ↔ Shop transfer
- Web dashboard (Flask)
- Power BI integration via CSV

## Tech Stack
Python, Flask, Pandas, ARIMA, Power BI

## How to Run
1. Install dependencies
2. Run data_generator.py
3. Run forecast_model.py
4. Run gap_analysis.py
5. Run optimize_reallocation.py
6. Run app.py
